```
+------------------------------------------+
|  Tournee en cours                        |
|  12 conteneurs — Prochain : #1          |
|                                          |
|  +------------------------------------+  |
|  |          CARTE                     |  |
|  |                                    |  |
|  |    (1)●        ●(4)               |  |
|  |         \      /                   |  |
|  |          (2)● /                    |  |
|  |            | /                     |  |
|  |           (3)●                     |  |
|  |            |                       |  |
|  |    ◉ ← vous etes ici              |  |
|  |                                    |  |
|  +------------------------------------+  |
|                                          |
|  [ Scanner QR ]  [ Incident ]           |
|                                          |
+------------------------------------------+
```

### Les couleurs sur la carte :

```
   ● VERT    = conteneur PLEIN     → collecter en PREMIER
   ● ORANGE  = conteneur A MOITIE  → collecter apres
   ● GRIS    = conteneur VIDE      → pas besoin
   ● BLEU    = VOUS (votre camion)
```

### L'ordre de collecte :

```
   Suivre les numeros !

   (1) → (2) → (3) → (4) → ...

   Le premier numero = le plus important
```

### Les boutons en bas :

```
   [ Scanner QR ]     → Arriver au conteneur
   [ Incident ]       → Signaler un probleme
   [ Retour centre ]  → Retour au depot
```

### Comment choisir :

```
   Regardez DANS le conteneur.

   1/2      →  a moitie             → toucher "1/2"
   3/4      →  presque plein        → toucher "3/4"
   Plein    →  completement plein   → toucher "Plein"
```

# 7. PESER LA COLLECTE

---

**Quand vous revenez au centre, vous pesez le camion.**

```
+------------------------------------------+
|  Pesee du vehicule                       |
|                                          |
|          +-------+                       |
|          | PESEE |                       |
|          +-------+                       |
|                                          |
|  Poids brut (kg)                        |
|  [ camion CHARGE : ex. 5200 ]           |
|                                          |
|  Tare vehicule (kg)                     |
|  [ camion VIDE : ex. 3500 ]             |
|                                          |
|  +------------------------------------+  |
|  |  Poids net : 1700 kg              |  |
|  |  (ce que vous avez collecte)       |  |
|  +------------------------------------+  |
|                                          |
|  +------------------------------------+  |
|  |      VALIDER LA PESEE             |  |
|  +------------------------------------+  |
+------------------------------------------+
```

### Comment faire :

```
1. Peser le camion CHARGE sur la balance
   → Ecrire le nombre dans "Poids brut"

2. Le poids du camion VIDE (tare)
   → Ecrire le nombre dans "Tare vehicule"
   → C'est ecrit sur le camion ou demander

3. L'application calcule :
   CHARGE - VIDE = ce que vous avez collecte

4. Verifier le chiffre

5. Appuyer sur  VALIDER LA PESEE
```

### Exemple :

```
   Camion charge  =  5200 kg
   Camion vide    =  3500 kg
                    --------
   Poids collecte =  1700 kg  ← c'est ca !
```

# 8. SIGNALER UN PROBLEME

---

**A tout moment, vous pouvez signaler un probleme.**

### Sur la carte, appuyer sur le bouton "Incident"

```
+------------------------------------------+
|  Signaler un incident                    |
|                                          |
|  Quel type de probleme ?                |
|                                          |
|  +------------------+------------------+ |
|  | Panne vehicule   | Accident         | |
|  +------------------+------------------+ |
|  | Conteneur / CAV  | Environnement    | |
|  +------------------+------------------+ |
|  | Autre            |                  | |
|  +------------------+------------------+ |
|                                          |
|  Description :                           |
|  [ ecrire ce qui s'est passe ]          |
|                                          |
|  +------------------------------------+  |
|  |       SIGNALER L'INCIDENT         |  |
|  +------------------------------------+  |
+------------------------------------------+
```

### Que faire pour chaque probleme :

---

**PANNE VEHICULE**

```
   1. Se garer en securite
   2. Mettre le gilet + triangle
   3. Sur l'application → "Panne vehicule"
   4. Ecrire ce qui ne marche pas
   5. Appuyer "Signaler l'incident"
   6. Appeler le responsable : attendre les instructions
```

**ACCIDENT**

```
   1. Ne pas bouger le vehicule
   2. Verifier si quelqu'un est blesse
   3. Si blesse → Appeler le 15 (SAMU) ou 18 (pompiers)
   4. Sur l'application → "Accident"
   5. Ecrire ce qui s'est passe
   6. Appuyer "Signaler l'incident"
   7. Appeler le responsable TOUT DE SUITE
```

**CONTENEUR CASSE**

```
   1. Sur l'application → "Conteneur / CAV"
   2. Ecrire : "conteneur casse" + ce qui est casse
   3. Appuyer "Signaler l'incident"
   4. Continuer la tournee
```

**CONTENEUR INACCESSIBLE**

```
   1. Sur l'application → "Conteneur / CAV"
   2. Ecrire : "conteneur inaccessible" + pourquoi
      (voiture garee devant, travaux, etc.)
   3. Appuyer "Signaler l'incident"
   4. Passer au conteneur suivant
```

**CONTENEUR ABSENT / VOLE**

```
   1. Sur l'application → "Conteneur / CAV"
   2. Ecrire : "conteneur absent" ou "conteneur vole"
   3. Appuyer "Signaler l'incident"
   4. Passer au conteneur suivant
```

# 10. EN CAS DE PROBLEME AVEC L'APPLICATION

---

### L'application ne repond plus ?

```
   1. Fermer le navigateur
   2. Rouvrir le navigateur
   3. Aller sur m.solidata.online
   → Votre tournee reprend ou vous etiez
```

### Pas de connexion internet ?

```
   1. Regarder en haut du telephone
      → Barres de signal ? Wi-Fi ?
   2. Si pas de signal :
      → Bouger un peu
      → Attendre
      → Essayer encore
   3. Si toujours rien :
      → Appeler le responsable
```

### L'ecran est bloque ?

```
   1. Fermer TOUT le navigateur
   2. Attendre 5 secondes
   3. Rouvrir le navigateur
   4. Aller sur m.solidata.online
```

### Le mot de passe ne marche pas ?

```
   1. Verifier les MAJUSCULES
      → Le mot de passe est sensible
   2. Si toujours bloque :
      → Appeler le responsable
      → Il peut reinitialiser le mot de passe
```

### Qui appeler ?
